## üìò Project Overview

The **IoT Greenhouse** is a smart greenhouse designed for the remote management of small plants. It integrates sensors, actuators, and a web interface to monitor and control environmental conditions efficiently and sustainably.

This project combines **informatics, electronics, and mechatronics** to create an accessible, educational, and eco-friendly system.

---

## üéØ Objectives

* Enable **remote control and monitoring** of the greenhouse via a web interface.
* Include an **alarm and environmental control system** for safety and automation.
* Promote **sustainable plant cultivation** through optimized water and electricity usage.
* Use **accessible hardware and open-source technologies** such as **Node.js**, **React**, and **Raspberry Pi**.
